## Background

**Galvanize Reads FrontEnd** is a book catalog service that stores a list of recommended technology books. You are building a web app for them. It should allow you to:

* List books and authors
* Add books and authors
* Modify books and authors
* Remove books and authors

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
